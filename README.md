**Si**m**p**le stateless **VPN**.

## Build ##

1. configure and make

	```bash
	autoreconf -if
	./configure --prefix=/usr --sysconfdir=/etc
	make
	```

2. install

	```bash
	sudo make install
	```

## Cross compile ##

1. setup cross compile tool chain

2. build

	```bash
	autoreconf -if
	./configure --host=arm-unknown-linux-gnueabihf \
	    --prefix=/usr --sysconfdir=/etc
	make
	```
